# Umme Kalsoom's Professional Portfolio 🌐

A modern, responsive, and visually appealing portfolio website for **Umme Kalsoom**, a 2nd Semester Computer Engineering student at **UET Lahore (Faisalabad Campus)**.

## 🚀 Features
- **Modern UI Design**: Built with high-end CSS, including glassmorphism, smooth animations, and a sleek Slate/Teal color palette.
- **Fully Responsive**: Optimized for desktop, tablets, and mobile devices.
- **Section-based Layout**: Smooth scrolling between Home, About, Skills, Projects, and Contact.
- **Interactive Elements**: Hover effects, scroll-reveal animations, and an active navigation menu highlight.
- **Tech Stack Focus**: Highlights skills in C++, Python, MySQL, and AI/ML.

## 📦 How to Use
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. To customize, edit the content directly in `index.html`.
4. To deploy: Push the files to your GitHub repository and enable **GitHub Pages** in the repository settings.
